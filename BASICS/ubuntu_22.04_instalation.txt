#########################################
 GUIDE TO INSTALL KOTLIN IN UBUNTU 22.04
#########################################

1- Update your package list 
    sudo apt update

2- Install Kotlin Using SDKMAN 
    -First install SDKMAN (tool to manage parallel versions of multiple SDKs):
    curl -s "https://get.sdkman.io" | bash source "$HOME/.sdkman/bin/sdkman-init.sh"
    -Then install kotlin 

3- Verify installation 
kotlin -version
